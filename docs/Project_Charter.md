# GyoolWork 기획서

## 1. 프로젝트 개요 & 배경

### 1-1. 개요

- 프로젝트명: GyoolWork
- 목적:
  - 별도의 학습없이 바로 사용할 수 있는 직관적인 협업 플랫폼 개발
  - 작업의 생성, 분배, 진행 현황, 성과 공유를 한 눈에 파악할 수 있도록 지원
  - 프로젝트 생성부터 문서 작성까지 한 번에 할 수 있는 플랫폼
- 핵심 기능:
  1. 효율적인 회사 구성 및 프로젝트 팀 구성
  2. 프로젝트 생성, 진행상황 공유
  3. 커뮤니케이션 도구(알림 및 채팅방)
  4. 문서 생성

- 기술 스택:
  - backend: Java, SpringBoot
  - frontend: Vue3
  - DB/캐시: MySQL, Redis
